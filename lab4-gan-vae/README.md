# Лабораторная 4 — GAN + VAE для восстановления изображений (MNIST)

Трёхкомпонентная нейросеть — гибрид VAE и GAN, реализованная полностью с нуля на чистом NumPy: свой автодифференцируемый слой `Linear` (Glorot-инициализация, ручной backward), свои функции активации и функции потерь.

## Архитектура

- **Encoder** (784 → 512 → 256 → 2×32): выдаёт параметры латентного распределения `μ` и `logσ²`
- **Decoder** (32 → 256 → 256 → 784): восстанавливает изображение из точки латентного пространства через reparameterization trick (`z = μ + σ·ε`)
- **Discriminator** (784 → 256 → 128 → 1): отличает реальные изображения от реконструированных декодером

## Функция потерь

Комбинация трёх составляющих:
- **Reconstruction loss** (BCE между входом и реконструкцией)
- **KL-дивергенция** между апостериорным распределением энкодера и стандартным нормальным (с warm-up коэффициентом `β`, линейно растущим первые 10 эпох — чтобы модель сначала научилась реконструировать, а потом регуляризовала латентное пространство)
- **Adversarial loss** — decoder дообучается «обманывать» дискриминатор, что подталкивает реконструкции к более реалистичным, менее «размытым» (характерная проблема чистого VAE)

Дискриминатор обучается отдельным шагом на реальных и сгенерированных изображениях (обычный GAN-лосс).
